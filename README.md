# Nationwide Erectors Website

This is the official website for Nationwide Erectors, a premier steel erection company specializing in commercial and industrial projects across the USA.

## Tech Stack

- **React**: UI library
- **TypeScript**: Type safety
- **Tailwind CSS**: Styling (via CDN)
- **Lucide React**: Icons
- **Vite**: Build tool

## Deployment

This project is configured to run on Netlify.

**Netlify Settings:**
- **Build command:** `npm run build`
- **Publish directory:** `dist`

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run local server:
   ```bash
   npm run dev
   ```
